# Gym Tracker

PWA for tracking gym workouts with GitHub sync.

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

This builds the app and pushes to the `gh-pages` branch.

### GitHub Pages Setup

1. Go to repo Settings > Pages
2. Source: Deploy from a branch
3. Branch: `gh-pages` / `/ (root)`
4. Save

Live at: https://nt1998.github.io/gym-tracker
